## EazyHash

### What is EazyHash
EazyHash is a simple python script that helps generate hash strings. Hash is available in 14 different hashing algorithms

## Available Hash Algorithms
* Shake_128
* sha3_256
* shake_256
* sha256
* md5
* blake2s
* blake2b
* sha3_384
* sha512
* sha3_224
* sha224
* sha3_512
* sha384
* sha1
